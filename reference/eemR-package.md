# eemR: Tools for Pre-Processing Emission-Excitation-Matrix (EEM) Fluorescence Data

Provides various tools for preprocessing Emission-Excitation-Matrix
(EEM) for Parallel Factor Analysis (PARAFAC). Different methods are also
provided to calculate common metrics such as humification index and
fluorescence index.
